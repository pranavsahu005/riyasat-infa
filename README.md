- **Hosting Ready**: Pre-configured for Netlify with SPA redirection support.

## Deployment Instructions

### Local Environment Setup

1. Clone or extract the project folder.
2. Run `npm install` to install dependencies.
3. Run `npm run build` to generate the production `dist` folder.

### Netlify Deployment

1. Log in to your [Netlify Dashboard](https://app.netlify.com/).
2. Drag and drop the `dist` folder (or the `riyasat-production.zip`) directly into the deployment area.
3. The site will be live instantly. All routes are handled by the `_redirects` file included in the build.
